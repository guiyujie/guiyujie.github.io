btn（按钮）

按钮样式。

标签

常规下，作为按钮的 HTML 标签有：

<button class="btn" type="button">button</button>
<input class="btn" type="submit" value="button"/>
<a class="btn" href="#">button</a>

button

皮肤

按钮皮肤。可选值有：

  • btn-blue 蓝色
<a class="btn" href="#">button</a>
<a class="btn btn-blue" href="#">button</a>

button button

大小

按钮大小。可选值有：

  • btn-mini 微型的
  • btn-smallest 最小的
  • btn-small 小的
  • btn-large 大的
  • btn-largest 最大的
<a class="btn btn-mini" href="#">mini</a>
<a class="btn btn-smallest" href="#">smallest</a>
<a class="btn btn-small" href="#">small</a>
<a class="btn" href="#">default</a>
<a class="btn btn-large" href="#">large</a>
<a class="btn btn-largest" href="#">largest</a>

mini smallest small default large largest
