ST（基础框架）

JS框架的使用

  1. 在 </head> 之前引入ST配置。
  2. 在 </body> 之前引入ST框架并初始化。
    <!doctype html>
    <html>
      <head>
        <meta charset="utf-8">
        <title>ST JavaScript Framework User Example</title>
        <!--引入jQuery库-->
        <script type="text/javascript" src="__YOURPATH__/js/jquery-min.js"></script>
        <!--ST配置-->
        <script>
          var ST = {
            // 资源路径
            PATH: {
              JS: "__YOURPATH__/js/",           //JS路径
              JSTMP: "__YOURPATH__/jsTemplate/",//JS模板路径
              IMAGE: "__YOURPATH__/images/",    //图片路径
              CSS: "__YOURPATH__/css/",         //CSS路径
              PUBLIC: "__YOURPATH__/",
              ROOT: "",                         //网站根目录
              VCODE: "",                        //验证码获取地址
              EDITORCSS:''                      //百度编辑器CSS样式路径
            },
            // 数据缓存
            CACHE: {},
            // JS模板
            JSTMP:{},
            // 服务端使用的URL
            ACTION: {
              //上传地址
              UPLOAD: "",
              //百度编辑器上传地址
              UPLOADUEDITOR: ""
            },
            //百度编辑器对象
            EDITORS:[],
            // 服务端输出的数据
            PHPDATA: {},
            // 初始化后执行的方法（ST.todoList()前执行）
            TODOLIST: [],
            // 页面空方法（用于HTML中执行ST方法，调用：ST.todo('方法名',参数1,...,参数N)）
            todo: function () {
            }
          };
        </script>
      </head>
      <body>
        ...
        <!--引入ST框架-->
        <script src="__YOURPATH__/js/ST.js"></script>
        <!--引入语言包-->
        <script src="__YOURPATH__/js/ST.LRes.js"></script>
        <!--引入ST插件（可选）-->
        ...
        <script>
        // 引入页面需要的JS模板（不支持跨域）
        ST.jsTemplates.push('common');
        ...
        // 扩展页面请求url（可选）
        $.extend(ST.ACTION,{
          ...
        });
        // 扩展页面PHP输出数据（可选）
        $.extend(ST.PHPDATA,{
          ...
        });
        // 扩展页面使用的JS模板（可选）
        $.extend(ST.JSTMP,{
          ...
        });
        // 扩展ST对象
        $.extend(ST,{
          // ST初始化后执行的方法
          todoList: function(){
            ...
          }
          // 自定义ST方法，调用 ST.myMethod(myPars);
          //,myMethod:function(myPars){//do sth.}
        });
        // 初始化 ST 框架
        ST.init();
        </script>
      </body>
    </html>

getJSON（ajax数据通信）

通用ajax获取JSON数据的方法。

调用方法

    ST.getJSON(
      '__URL__',    //服务端URL
      {},           //发送的数据
      function(){   //成功后的回调方法
        //do sth.
      },
      function(){   //失败后的回调方法
        //do sth.
      },
      'post',       //HTTP方法（"get"|"post"，默认"post"）
      'json'        //数据类型（"json"|"jsonp"，默认"json"）
    );

数据接口

      {
        "status":"",        //通信状态
        "message":"",       //附加信息（或"info":""）
        "data":"",          //附加数据
      }
      /*
        常用通信状态：
        success       成功
        tip_success   成功，使用tipMsg提示附加信息中的内容
        error         失败
        tip_error     失败，使用tipMsg提示附加信息中的内容
      */

Spinner（数值微调器）

Browser Support

浏览器支持,已在以下浏览器版本中测试通过。

  • 27
  • 22
  • 7+

HTML

    <input id="_spinner" type="text">

CSS（LESS）

    @import "LESS/component/spinner/spinner";

JavaScript

    new ST.Spinner("_spinner");

带选项（参数）使用

    new ST.Spinner("_spinner1",{
      increment: 0.1,
      prefix: '共',
      suffix: '元',
      inputClass:'input-smallest'
    });

Options

Name Type Default Description
_default number 0 默认值
min number 0 最小值
max number 100 最大值
increment number 1 增量
prefix string none 前缀文本
suffix string none 后缀文本
width string input-mini 组件宽度，可选值为 input-mini、input-smallest、input-small、input、
input-large、input-largest
template string common_spinner 根据名称选择 ST 模板
inputable boolean true 是否可输入
disabled boolean false 是否禁用
keyboard boolean true 是否支持上下方向键调节
mousewheel boolean false 是否支持鼠标上下滚动调节

PS：支持鼠标滚动调节需另外引入 jquery.mousewheel.js 文件并将 mousewheel 参数值设置为 true 。

Methods

Name Description
.add() 增加单元increment
.discount() 减小单元increment
.getNumValue() 获取当前数值
.getValue() 获取当前显示值
.enable() 启用
.disable() 禁用
.onChange() ...

CityList（区省市县联动，单选）

（基于ddList实现，需服务端提供数据）

数据接口

  {
    //大区数据（空数据：alist:null）
    "alist":[
      {
        "id":"1",       //大区id
        "name":"大区1"  //大区名称
      }
      //,……
    ],
    //省份数据（空数据：plist:null）
    "plist":{
      "1":[             //所属大区id
        {
          "id":"1",     //省份id
          "name":"省份1"//省份名称
        }
        //,……
      ]
      //,……
    },
    //城市数据（空数据：clist:null）
    "clist":{
      "1":[             //所属省份id
        {
          "id":"1",     //城市id
          "name":"城市1"//城市名称
        }
        //,……
      ]
      //,……
    },
    //区县数据（空数据：xlist:null）
    "xlist":{
      "1":[             //所属城市id
        {
          "id":"1",     //区县id
          "name":"区县1"//区县名称
        }
        //,……
      ]
      //,……
    }
}

用法

  1. 引入ST.CityList.js
  2. 引入从服务器获取的数据（符合以上数据接口）
  3. See more...
