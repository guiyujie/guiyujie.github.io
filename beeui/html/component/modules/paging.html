paging（分页）

分页组件样式。

结构

<div class="paging">
    <ul>
        <li class="prev"><a href="#"><i class="icon icon-prev"></i></a></li>
        <li class="active"><a href="#">1</a></li>
        <li><a href="#">2</a></li>
        <li><a href="#">3</a></li>
        <li class="next"><a href="#"><i class="icon icon-next"></i></a></li>
    </ul>
</div>

大小

beeui 提供三种不同大小的分页组件：

  • paging-smt
  • paging-sm
  • paging

对齐

beeui 提供不同的对齐方式支持。

  • paging
  • paging-centered
  • paging-right
<div class="paging paging-centered">
    <ul>
        <li class="prev"><a href="#"><i class="icon icon-prev"></i></a></li>

        <li><a href="#">3</a></li>
