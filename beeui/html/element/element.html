btn（按钮）

按钮样式。

标签

常规下，作为按钮的 HTML 标签有：

<button class="btn" type="button">button</button>
<input class="btn" type="button" value="button"/>
<a class="btn" href="javascript:;">button</a>

button

形态

按钮展现形态。beeui 提供三种不同形态的按钮：

<a class="btn" href="javascript:;">button</a>
<a class="btn btn-unbent" href="javascript:;">unbent button</a>
<a class="btn btn-block" href="javascript:;">block button</a>

button unbent button

block button

皮肤

按钮皮肤。beeui 中按钮皮肤可选值有：

  • btn 默认（橙色）
  • btn-blue 蓝色
  • btn-green 绿色
  • btn-gray 灰色
  • btn-white 白色
<a class="btn" href="javascript:;">default button</a>
<a class="btn btn-blue" href="javascript:;">blue button</a>
<a class="btn btn-green" href="javascript:;">green button</a>
<a class="btn btn-gray" href="javascript:;">gray button</a>
<a class="btn btn-white" href="javascript:;">white button</a>

default button blue button green button gray button white button

状态

按钮状态。常用状态有： default（默认）、hover、active、disabled

<a class="btn" href="javascript:;">default button</a>
<a class="btn hover" href="javascript:;">blue button</a>
<a class="btn active" href="javascript:;">green button</a>
<a class="btn disabled" href="javascript:;">gray button</a>
...

btn hover active disabled

btn hover active disabled

btn hover active disabled

btn hover active disabled

btn hover active disabled

大小

按钮大小。beeui 为按钮提供 6 种不同的大小值：

  • btn-mn 微型的
  • btn-smt 最小的
  • btn-sm 小的
  • btn 默认
  • btn-lg 大的
  • btn-lgt 最大的
<a class="btn btn-mn" href="javascript:;">mini button</a>
<a class="btn btn-smt" href="javascript:;">smallest button</a>
<a class="btn btn-sm" href="javascript:;">small button</a>
<a class="btn" href="javascript:;">button</a>
<a class="btn btn-lg" href="javascript:;">large button</a>
<a class="btn btn-lgt" href="javascript:;">largest button</a>

mini button smallest button small button button large button largest button

form（表单）

表单样式。

form-inline

内联表单样式。

<form class="form-inline" action="" method="">
    <label for="n1">name</label>
    <input class="input" type="text" id="n1"/>

    <label for="p1">password</label>
    <input class="input" type="password" id="p1"/>

    <label class="checkbox" for="cb1">
        <input type="checkbox" id="cb1"/>
        remember?
    </label>

    <button class="btn" type="submit">submit</button>
</form>

form-v

垂直布局表单样式。

<form class="form-v" action="" method="">

    <div class="row">
        <label class="item" for="n2"><span class="required">*</span>name</label>

        <div class="span">
            <input class="input" type="text" id="n2"/>
            <div class="help-block">help-block</div>
        </div>
    </div>

    <div class="row">
        <label class="item" for="p2">password</label>

        <div class="span">
            <input class="input" type="password" id="p2"/>
            <div class="help-inline">help-inline</div>
        </div>
    </div>

    <div class="row form-actions">
        <div class="span">
            <button class="btn" type="submit">提交</button>
            <button class="btn btn-gray" type="button">取消</button>
        </div>
    </div>

</form>
