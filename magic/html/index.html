magic 简介

      magic：一个基于 cosyless 的 CSS3 动画库。

使用方式

作为独立 CSS 文件引入 html 页面。

      <link rel="stylesheet" href="../dest/css/magic-desktop.min.css"/>

文件可选值有：

  • magic.min.css：Webkit版（CSS3属性兼容 W3C、-webkit-）。【42kb】
  • magic-desktop.min.css：PC版（CSS3属性兼容 W3C、-webkit-、-moz-）。【83kb】
  • magic-mobile.min.css：Mobile版（CSS3属性兼容 W3C、-webkit-、-o-、-ms-、-moz-）。【205kb】

html 页面引入 magic 动画库后，给定元素添加动画基类 magic。

      <div class="magic"></div>

选择追加所需的动画效果类名，就会具有对应的 CSS 动画效果，如：shake。

      <div class="magic shake"></div>

若需要无限循环动画，添加 infinote 类即可。

      <div class="magic shake infinote "></div>
