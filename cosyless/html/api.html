fix

常用 CSS 浏览器 hack 样式（修复）

clearfix

清除浮动

      .clearfix {

      //  IE 8及以上、Chrome、Firefox 使用 伪类 清除浮动
        &:before,
        &:after {
          content: " ";
          display: table;
        }

        &:after {
          clear: both;
        }

      //  IE 6 ~ IE 7 触发 IE hasLayout 清除浮动
        *zoom: 1;
      }

使用示例：

      // 用法一
      class="clearfix"

      // 用法二
      .example { .clearfix(); }

inline-block

行块元素转换

      .inline-block(@marginRight: 3px) {
      //  IE 8 及以上
        display: inline-block;
      //  IE 6 ~ IE 7（不支持行块显示，转换为行内元素并添加右外边距，尽可能保持行块元素跨浏览器显示一致）
        *display: inline;
        *zoom: 1;

        *margin-right: @marginRight;
      }

使用示例：

    .example { .inline-block(); }

minfix

最小宽高值转换

      .min-width(@width) {
      //  IE 8 及以上
        min-width: @width;
      //  IE 6 ~ IE 7（针对 IE7 hack）
        width: auto !important;
      //  IE 6 ~ IE 7（针对 IE6 hack）
        width: @width;

      //  方案二
      //  min-width: @width;
      //  _width: @width;
      }

      .min-height(@height) {
        min-height: @height;
        height: auto !important;
        height: @height;

      //  方案二
      //  min-height: @height;
      //  _height: @height;
      }

使用示例：

      .example { .min-width(400px); }

transparent

背景色透明转换

      .transparent(@r, @g, @b, @a) {
        @argb: argb(rgba(@r, @g, @b, @a));

        background: e('rgb(@{r}, @{g}, @{b})') transparent;
      //  For IE 9 及以上
        background: rgba(@r, @g, @b, @a);

      //  For IE 5.5 ~ IE 7
        filter: e('progid:DXImageTransform.Microsoft.gradient(startColorstr=@{argb}, endColorstr=@{argb})');
      //  For IE 8 ~ IE 9
        -ms-filter: e('"progid:DXImageTransform.Microsoft.gradient(startColorstr=@{argb}, endColorstr=@{argb})"');
      }

使用示例：

      .example { .transparent(0, 0, 0, .5); }

border-radius

CSS3 边框圆角

      .border-radius(...) {

          & when (@type = mobile) {
              -ms-border-radius: @arguments;
              -o-border-radius: @arguments;
          }

          & when (@type = desktop) {
              -moz-border-radius: @arguments;
          }

          -webkit-border-radius: @arguments;
          border-radius: @arguments;
      }

使用示例：

      .example { .border-radius(4px 3px 2px 1px); }

生成结果：

      // @type = webkit
      .example {
        -webkit-border-radius: 4px 3px 2px 1px;
        border-radius: 4px 3px 2px 1px;
      }

      // @type = desktop
      .example {
        -moz-border-radius: 4px 3px 2px 1px;
        -webkit-border-radius: 4px 3px 2px 1px;
        border-radius: 4px 3px 2px 1px;
      }

      // @type = mobile
      .example {
        -ms-border-radius: 4px 3px 2px 1px;
        -o-border-radius: 4px 3px 2px 1px;
        -moz-border-radius: 4px 3px 2px 1px;
        -webkit-border-radius: 4px 3px 2px 1px;
        border-radius: 4px 3px 2px 1px;
      }

box-shadow

CSS3 盒阴影

      .box-shadow(...) {

          & when (@type = mobile) {
              -ms-box-shadow+: @arguments;

          }

          & when (@type = desktop) {
              -moz-box-shadow+: @arguments;
          }

          -webkit-box-shadow+: @arguments;
          box-shadow+: @arguments;
      }

使用示例：

    .example {
        .box-shadow(4px 3px 2px #fff);
        .box-shadow(inset 4px 3px 2px #fff);
